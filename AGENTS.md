# Project Instructions

- Общайся с пользователем на русском языке.
- Работай самостоятельно и доводи запрошенные задачи до готового результата.
- После любых изменений файлов в этом проекте обязательно создай Git-коммит с понятным сообщением.
- Изменения сайта не считаются завершёнными после одного локального коммита: после проверки отправь коммит в `origin/main`, дождись успешной публикации GitHub Pages и проверь изменения на `https://bite-studio.ru`. При изменении CSS, JavaScript или медиа обнови версию URL ресурса, чтобы браузеры не показывали старый кэш.
- Не проси пользователя выполнять коммит вручную.
- Для операций с GitHub используй уже настроенную локальную аутентификацию; не сохраняй секреты, пароли или токены в репозитории.
- Все версии видео для сайта, включая мобильные и оптимизированные копии, должны сохранять частоту 60 FPS. Снижать видео до 30 FPS запрещено.
